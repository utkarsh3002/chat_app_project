## Installation

First, clone this repository:

# Clone this repository
git clone https://github.com/utkarsh3002/chat_app_project.git
cd chat_app_project

# Install dependencies
npm install

# Run the backend
cd nodeserver
nodemon ./index.js

# Open http://localhost:5500 in your browser
open http://localhost:5500
